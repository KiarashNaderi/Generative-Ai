# AI Business Strategist (LangChain + Streamlit)

An AI-powered business strategist chatbot built with LangChain and Streamlit.  
This project helps founders, entrepreneurs, and product builders think through their ideas, refine strategies, and get structured business advice using large language models.

---

## Features

- AI-powered business strategy assistant
- Chat-based interface with Streamlit
- Conversation memory to keep context across messages
- Modular architecture (LLM, prompts, memory, strategist logic separated)
- Built with modern LangChain (LCEL / Runnable pipeline)
- Easy to extend with RAG, tools, or agents
